# Modern Music Player

A beautiful, cinematic web-based music player with dark mode support and custom file upload functionality. This lightweight application allows users to organize and play their favorite songs directly in the browser without any server requirements. Experience smooth playback controls, stunning visual effects, and persistent playlist management.

## Key Features

- **Upload & Play Custom Songs** - Add your own MP3 files to create a personalized playlist that persists across sessions using browser local storage
- **Dark Mode Toggle** - Switch between elegant light and cinematic dark themes with smooth animations and automatic preference saving
- **Full Playback Controls** - Play, pause, skip tracks, adjust volume, seek through songs, and view album artwork with an intuitive interface

## Technologies

- **HTML5** - Semantic markup and audio element API
- **CSS3** - Modern styling with gradients, animations, and glassmorphism effects
- **JavaScript (Vanilla)** - Pure JavaScript for all functionality without external frameworks
- **Font Awesome Icons** - Beautiful icon library for UI elements
- **Local Storage API** - Browser-based data persistence for playlists and preferences
